# BOILERPLATE REACT REDUX SAGA TYPESCRIPT

É um projeto default para criar novos projetos nessas tecnologias citadas no título

## Tecnologias usadas

- react
- typescript
- redux
- redux sagas
- axios
- typesafe actions
- eslint
- prettier

## Instalação

`git clone git@github.com:diegohyenna/boilerplate-react-ts-redux-saga.git`

na pasta raíz
`npm install`
